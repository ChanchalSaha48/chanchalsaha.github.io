Dual-core silver-coated plasmonic sensor modeling using machine learning techniques

Authors: Chanchal Saha (First Author), Others

Journal: Heliyon (Elsevier, CellPress, Q1, Scopus & Web of Science Indexed)

Year: 2024

Citations: 14 (Google Scholar)

This work presents an intelligent modeling approach for dual-core silver-coated plasmonic sensors using machine learning techniques. The study demonstrates strong predictive accuracy while reducing computational cost, showing the potential of ML-assisted sensor design.
